🚀 Quick Publish — Push Changes Live

Step 1: Open Terminal app on Mac
Step 2: Copy-paste this one line:
cd /Users/gk/Documents/website && git add -A && git commit -m "site update" && git push
✅ Done! Site updates in 30 seconds

🕐 Check Auto Jobs (Cron) Running

See all scheduled jobs:
hermes cron list
Run a job right now (test):
hermes cron run JOB_ID
Stop a job:
hermes cron remove JOB_ID

🛡️ Preview Site Locally (Before Publishing)

cd /Users/gk/Documents/website && python3 -m http.server 4000
Then open http://localhost:4000 in browser

🔁 Emergency — Undo Changes & Restore

⚠️ Use only if you broke something!
Restore last saved version from GitHub:
cd /Users/gk/Documents/website && git checkout -- .
Full reset to GitHub version (wipes local changes):
cd /Users/gk/Documents/website && git fetch origin && git reset --hard origin/main
